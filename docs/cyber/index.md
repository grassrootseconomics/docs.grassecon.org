# Software

Grassroots Economics software supports Community Asset Vouchers, Commitment Pools, reports, maps, wallets, and public data. This section gives practical entry points for users, stewards, and developers.

## Sarafu Network

[Sarafu Network](https://sarafu.network) is the live platform for creating and browsing vouchers, creating and managing Commitment Pools, viewing reports, using wallets, and exploring community activity. It is the main public interface for current Commitment Pooling work.

Common actions:

* Browse or create [vouchers](https://sarafu.network/vouchers).
* Browse or create [pools](https://sarafu.network/pools).
* View [reports](https://sarafu.network/reports) and community activity.
* Use the web wallet or, where available, USSD access.
